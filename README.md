# Cotton-Plant-Disease-Detection
### --------------------Algorithms I used:-----------------------
I used Convolutional Neural Networks (CNN).                                                   
Architecture: ResNet 50 ,VGG 16 and custom.

#### Custom Network: 
1.Convolutional Layer - 2                             
Max Pooling layer - 2                               
Flatten and Dense layer                             
Input size: 64*64                

2. Convolutional Layer - 4                   
Max Pooling layer - 3   
Flatten and Dense layer                         
Input size: 224*224
